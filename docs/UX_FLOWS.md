# UX Flows Specification

## Satcom Workforce Visibility System

---

## Design Principles

### Core UX Goals
1. **Speed**: Every interaction feels instant (< 200ms perceived)
2. **Clarity**: One primary action per screen
3. **Consistency**: Same patterns across web and mobile
4. **Accessibility**: WCAG 2.1 AA compliance

### Interaction Patterns
- Skeleton loading for all data fetches
- Optimistic updates for common actions
- Smooth transitions (200-300ms)
- Clear error states with recovery actions
- Pull-to-refresh on mobile lists
